# 计划：使用 grill-me 技能梳理项目目标

## 目标
通过 grill-me 技能的追问流程，帮助用户重新梳理「淘系店铺经营数据分析」项目的目标、方向和优先级。

## 执行步骤

### Step 1: 加载 grill-me 技能
- 使用 `skill` 工具加载 `grill-me` 技能
- 获取技能的完整指令和工作流程

### Step 2: 启动追问会话
- 基于项目当前状态（status.md、design.md）作为起点
- 从以下维度逐一追问：
  1. **项目定位**：这个项目最终要解决什么问题？目标用户是谁？
  2. **核心价值**：相比手动分析，这个工具带来的最大价值是什么？
  3. **功能边界**：哪些功能是必须的，哪些是锦上添花，哪些可以砍掉？
  4. **技术方向**：Excel 驱动流是否仍然是最佳选择？是否需要回归 API 驱动？
  5. **优先级**：当前待办清单中的项目，哪些应该优先做？
  6. **成功标准**：怎样算项目「完成」了？

### Step 3: 记录结论
- 将梳理后的目标写入 `docs/proposal.md` 或更新 `status.md`
- 如有关键决策，记录到 `ADR.md`

## 关键文件
- `status.md` — 当前项目状态
- `docs/design.md` — 当前架构设计
- `AGENTS.md` — 项目硬规则

## 验证方式
- 追问结束后，用户能清晰回答「这个项目要做什么、为什么做、先做什么」
- 产出文档反映梳理后的目标
