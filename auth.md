## 归档内容（历史参考）

原认证流程包含：
1. 订购「数仓」服务
2. 绑定需要分析的店铺
3. 获取请求码（Token Key）并配置 `auth/jycm.json`
4. Cookie 获取 / 刷新 / Digest 鉴权

详细流程见 Git 历史版本或原 `SKILL.md` v4.0.0。

---

## 当前版本说明

- **无需认证**：本技能当前版本直接读取本地 Excel，不涉及任何外部 API 调用
- **无 Cookie 依赖**：所有数据来自用户上传的文件
- **如需 API 版**：请使用 `qianniu_business_analytics` v4.x 或联系瓴羊·One 服务
